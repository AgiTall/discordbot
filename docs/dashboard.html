Каналы

Настройте каналы для разных функций бота

Канал новостей

Канал, в который бот публикует анонсы и новости. Настраивается командой /set-channel.

Канал для ежедневной раздачи карт сокровищ в 12:00 МСК.

Автоматические треды

Каналы, где бот автоматически создаёт треды к новым сообщениям. Команды: /threads-on и /threads-off.

Командные каналы

Каналы, где разрешены команды игроков. Добавить: /command-chat #канал. Удалить: /command-chat #канал remove:True.

Разрешить команды во всех каналах Если включено, список выше игнорируется

Настройки XP

Множители опыта для разных источников

Базовые значения

Настраивается командой /set-xp-rate <источник> <множитель>

Источник: messages. Базовый XP = 15 × множитель.

Источник: voice. Начисляется раз в минуту.

Источник: jobs. Работа и профессиональные действия.

Источник: events. Специальные мероприятия.

Дополнительный множитель к базовому XP за сообщения.

Дополнительный множитель к базовому XP в голосовых каналах.

Формула опыта

XP для уровня N = 100 × N1.5

Пример: уровень 5 требует 559 XP, уровень 10 — 3162 XP, уровень 20 — 8944 XP. Смотрите полную таблицу на странице Уровни.

Уровень 10: нужно 3 162 XP

Приветствия

Настройки приветственных сообщений для новых участников

Включить приветствия Отправлять сообщение при входе нового участника

Роль, которая автоматически выдаётся при входе.

Переменные: {mention} — упоминание, {user} — имя, {server} — название сервера, {count} — номер участника.
